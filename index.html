Restaurant Management System

Welcome to the Restaurant Management System API. This server provides REST endpoints for managing restaurant categories and menu items.

Note: This is the backend API server. The React frontend application should be running separately.

Available API Endpoints

Categories

  • GET /api/categories
    Get all categories
  • GET /api/categories/{id}
    Get a specific category by ID
  • POST /api/categories
    Create a new category
  • PUT /api/categories/{id}
    Update an existing category
  • DELETE /api/categories/{id}
    Delete a category

Menu Items

  • GET /api/menu-items
    Get all menu items
  • GET /api/menu-items/{id}
    Get a specific menu item by ID
  • GET /api/menu-items/category/{categoryId}
    Get menu items by category
  • POST /api/menu-items
    Create a new menu item
  • PUT /api/menu-items/{id}
    Update an existing menu item
  • DELETE /api/menu-items/{id}
    Delete a menu item

Getting Started

To use this API with the React frontend:

  1. Ensure this backend server is running (you're seeing this page, so it is!)
  2. Start the React frontend application
  3. The React app will connect to these API endpoints to manage restaurant data
